# Socket_Programming_Traditional

## Usage
 - compile
 
 ```gcc server.c -o s
    gcc client.c -o c
 ```
 
 - start server
  > server is waiting for clients.
 
 ```./s```

 - start client
  > open a terminal

```./c```
